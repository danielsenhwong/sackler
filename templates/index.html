Website to iCal Calendar Conversion

This is a Python script that parses the Tufts Sackler School website calendar RSS feed into an iCal format that can be subscribed to in Google Calendar, Apple Calendar, Microsoft Outlook, and other similar programs.

The information is re-formatted from the RSS feed into a more compact layout. In particular, the room number of an event is incorporated into the iCal Location information for an event in parentheses. This allows for human readability while maintaining compatability with Google Maps for mobile device users.

The script is scheduled to run and update all defined calendars at 4 AM daily. However, individual calendar programs such as Google Calendar may not reflect recent changes for a few hours. This is a known issue with Google Calendar (Google Product Support Forum).

Source code is available on GitHub: danielsenhwong/sackler.
